# Snowflake Vaccination Quality Guardian

AI Guardian monitors vaccination records in Snowflake, using Hybrid and Dynamic Tables plus Cortex AI to detect anomalies, explain issues in plain English, suggest fixes, and show real-time Streamlit dashboards.

## Features

- **Dashboard**: Real-time overview of data quality scores, issue counts, and trends.
- **Anomaly Detection**: Detailed breakdown of critical and warning issues with AI-powered explanations.
- **AI Summary**: Executive summary and recommended actions generated by Snowflake Cortex AI.

## Getting Started

### Prerequisites

- Python 3.8+
- [Streamlit](https://streamlit.io/)

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd snowflake-vaccination-quality
   ```

2. Create a virtual environment (optional but recommended):
   ```bash
   python -m venv venv
   # On Windows:
   venv\Scripts\activate
   # On macOS/Linux:
   source venv/bin/activate
   ```

3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

### Running the App

To start the Streamlit application, run:

```bash
streamlit run project.py
```

The application will open in your default web browser at `http://localhost:8501`.
